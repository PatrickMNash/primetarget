About

Prime Target is a math game. It is a relatively simple concept that proves more challenging than it might appear at first.

Game Play

The concept is fairly simple, subtract values from a starting value until a target value is reached, undershot, or until moves have been exhausted.

The Game Interface

The game interface is a simple 3x3 grid of colored squares.

The Meaning Of Each Square In The Grid

Square 1

Square 1 is the number of moves you have left in the current round. As you select values to subtract this number will decrease by 1.

As the level of the game increases the number of moves decreases until you reach level 7 and 3 moves. This is the minimal number of moves.

Square 2

Square 2 is the starting value and current value after performing a subtraction.

Square 3

Square 3 is your current score.

Squares 4-6

Squares 4, 5, and 6 are the available values you can subtract from Square 2.

Square 7

Square 7 is the current level. Every time the target value is reached the level increases by 1.

Square 8

Square 8 is the target value that must be reached in order to succesfullly complete a level.

Square 9

Square 9 is "hearts". Pressing Square 9 allows you to subtract 1 from Square 2. This does not count against Square 1 (moves remaining). Performing multiple decrements is allowed.

Losing Hearts

If a round ends without reaching the target (Square 8) then hearts are subtracted from the total in Square 9.

Until level 7 is reached 1 heart is subtracted per round if the target (Square 8) is missed. At level 7 the penalty increases to 2. Going forward when a level is reached which is a multiple of 7 the penalty increases by 1. Example: The penalty at level 49 would be: 49 / 7 = 7, 7 + 1 = 8 hearts deducted if the target is missed.

Gaining Hearts

When the target is reached and the level is advanced, 1 heart is added to the total in Square 9.

Game Over

The game is over when there are 0 hearts left in Square 9 and the round ends with the target (Square 8) being missed.
